# SaveDaDay - Birthday Reminder App

A MERN application with NextJS and TailwindCSS, which allows user to manage the birthdays of their friends and family.

## Tech Stack
- **Frontend**: `NextJs` + `TailwindCSS`
- **Backend**: `ExpressJs` + `MongoDB`
